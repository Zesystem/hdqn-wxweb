# 获取数据方法
+ code为200时获取成功
+ 后台返回json数据data
+ 假定一次性请求20条数据
+ 实际数据以count参数为准
+ 请求此数据的后n条数据为 {next=当前所在条数, count=n}

# 提交数据方法
+ 请求方式post
+ 请求数据需要id时，需要请求素材上传接口获取id
+ method为请求类型，类型列表['get', 'add', 'del']
+ type为资源类型
+ 素材类型列表[0, 1, 2, 3, 4, 5] 分别对应文本、图文、语音、视频、图片、音乐6个素材
+ 反馈类型列表[6, 7, 8] 分别对应河小博、河小知、河掌柜
+ 数据统计类型列表[9, 10] 分别对应关键词、接口 


```
// ====================================
// 查看反馈
// ====================================
// 河小博，河小知，河掌柜
// count为返回的条数，iterms为count条数据

# 请求
data = {
    'type': 6,
    'next' : 0,
    'count' : 20,
    'method' : 'get'
}

# 返回数据
data = {
    'code' : 200,
    'count'  : 20, 
    'iterms' : [   
        {
            'keyword' : ''，
            'content': ''
        },
        {
            'keyword' : ''，
            'content': ''
        }
        ....
    ]
}
```
